1 Configuração

1.1 Atalho - Portacle

https://portacle.github.io/ … e já está!

1.2 Senão, instalar todos os componentes

SBCL: o compilador

http://www.sbcl.org/

Emacs: o editor de texto programável

https://www.gnu.org/software/emacs/

Slime: uma "extensão" para o Emacs que faz a ligação com o SBCL

https://common-lisp.net/project/slime/

Quicklisp: o gestor de bibliotecas

https://www.quicklisp.org/beta/

Ferramentas instaladas a partir do Emacs

  • ParEdit: editar s-expressions

    https://www.emacswiki.org/emacs/ParEdit

  • Magit: trabalhar com o sistema de version control Git dentro do Emacs

    https://magit.vc/
